# Available regions for Google AI Studio and Gemini API

If you reached this page after trying to open [Google AI Studio](https://makersuite.google.com/), it may be because Google AI Studio is not available in your region, or you don't meet the age requirements (18+) for access. You can learn more about the available regions in the following section and other requirements in the [terms of service](/gemini-api/terms).

## Available regions

**Note:** For Colab users - Region restrictions are applied based on the region that the **Colab instance** is in, not the region that **the user** is in. You can check the location of the **Colab instance** using `!curl ipinfo.io`
